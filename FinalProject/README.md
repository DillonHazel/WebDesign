# Site Plan

## Home Page

**Goal:** Introduce the website and create a welcoming entry point for users exploring Montana.

### Structure
- **Navigation:** Bootstrap navbar linking to all pages (Home, Flexbox, Grid, Blog, Canvas)
- **Hero / Intro:** Welcome text and introduction to Montana theme
- **Main Content:** Short description about the purpose of the site
- **Supporting Content:** Social icons or visual branding elements
- **Footer:** Social links and site-wide navigation consistency

---

## Flexbox Page

**Goal:** Demonstrate layout skills using Flexbox through blog-style content cards.

### Structure
- **Navigation:** Consistent navbar across all pages
- **Hero / Intro:** Page title introducing Flexbox layout
- **Main Content:** Blog-style Flexbox layout with images and text previews
- **Supporting Content:** Card section demonstrating responsive layout
- **Call-to-Action:** Links to blog post page
- **Footer:** Social icons and navigation consistency

---

## Grid Page

**Goal:** Showcase CSS Grid layout techniques with a two-column design.

### Structure
- **Navigation:** Consistent navbar
- **Hero / Intro:** Featured section introduction
- **Main Content:** Grid-based blog previews and layout structure
- **Supporting Content:** Right column with hero image and cards
- **Footer:** Standard site footer with social links

---

## Blog Post Page

**Goal:** Provide a detailed article-style page expanding on Montana content.

### Structure
- **Navigation:** Site-wide navbar
- **Hero / Intro:** Blog title and introduction paragraph
- **Main Content:** Written blog content about Montana and future ideas
- **Supporting Content:** None or minimal for focus on reading experience
- **Call-to-Action:** Optional link back to blog or homepage
- **Footer:** Social icons and site navigation consistency

---

## Canvas Practice Page

**Goal:** Demonstrate JavaScript and HTML5 Canvas animation skills.

### Structure
- **Navigation:** Site-wide navbar
- **Hero / Intro:** Simple page title introducing canvas animation
- **Main Content:** Animated canvas drawing (glowing circle animation)
- **Supporting Content:** Optional explanation or minimal layout
- **Footer:** Social icons and consistent site footer
